Analisis de Desempeño Comercial - Alura Store Latam

Descripcion del Proyecto
Este proyecto consiste en un analisis de datos integral para una cadena de cuatro tiendas pertenecientes a Alura Store. El objetivo principal es evaluar el rendimiento de cada sucursal mediante indicadores clave de desempeño (KPIs) para recomendar al propietario, el Sr. Juan, cual de las tiendas debe ser vendida para reinvertir el capital en un nuevo negocio.

Estructura de los Datos
El analisis se realizo utilizando cuatro fuentes de datos distintas correspondientes a cada una de las tiendas. Los datos incluyen informacion sobre:

Productos vendidos y sus categorias.

Precios de venta unitarios.

Costos de envio por transaccion.

Calificaciones de satisfaccion del cliente.

Ubicacion geografica (Latitud y Longitud) y metodos de pago.

Metodologia de Analisis
El flujo de trabajo se dividio en las siguientes etapas tecnicas:

Extraccion y Carga: Importacion de archivos CSV y JSON desde fuentes remotas.

Procesamiento: Consolidacion de datos y limpieza de tipos de variables.

Analisis de Facturacion: Calculo de ingresos totales por punto de venta.

Analisis de Producto: Identificacion de volumen de ventas por categoria y productos estrella.

Evaluacion de Calidad: Calculo de promedios de satisfaccion del cliente.

Analisis Logistico: Determinacion del costo de envio promedio por tienda.

Hallazgos y Resultados
Tras procesar la informacion, se identificaron los siguientes puntos criticos:

Ingresos: Se determino que tiendas mantienen el mayor flujo de caja.

Logistica: Se hallaron diferencias significativas en los costos de envio, lo cual afecta la conversion de ventas.

Satisfaccion: Se detectaron variaciones en las calificaciones, identificando unidades con problemas operativos o de servicio.

Conclusion y Recomendacion Tecnica
Basado en el cruce de variables, se emitio una recomendacion final fundamentada en datos (Data-Driven). La tienda seleccionada para la venta es aquella que presenta la combinacion de menor satisfaccion del cliente y mayores costos logisticos, representando el activo menos eficiente para la cadena.
